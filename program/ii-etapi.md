# II рЃћрЃбрЃљрЃърЃў

рЃњрЃўрЃџрЃЮрЃфрЃљрЃЋ ­ЪјЅ рЃЌрЃБ рЃљрЃЏрЃљрЃА рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉ рЃћрЃАрЃћрЃўрЃњрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЕрЃћрЃџрЃћрЃюрЃ»рЃў рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃЊрЃљрЃљрЃАрЃарЃБрЃџрЃћ рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃ«рЃЋрЃћрЃЊрЃў рЃерЃћрЃЏрЃЊрЃћрЃњ рЃћрЃбрЃљрЃърЃќрЃћ ­Ъџђ

\(рЃЌрЃБ рЃљрЃа рЃЊрЃљрЃњрЃўрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃљ рЃљрЃю рЃљрЃарЃф рЃЊрЃљрЃњрЃўрЃгрЃДрЃўрЃљ рЃЏрЃљрЃерЃўрЃю рЃАрЃ«рЃЋрЃљрЃњрЃљрЃю рЃЏрЃЮрЃ«рЃЋрЃЊрЃў рЃЊрЃљ рЃ»рЃЮрЃЉрЃўрЃљ [рЃљрЃЦ рЃерЃћрЃЏрЃЮрЃњрЃЋрЃўрЃћрЃарЃЌрЃЊрЃћ](i-etapi.md)\)

рЃЏрЃћрЃЮрЃарЃћ рЃћрЃбрЃљрЃърЃќрЃћ рЃЏрЃЮрЃАрЃљрЃгрЃЋрЃћрЃЋрЃўрЃА рЃЏрЃўрЃАрЃљрЃдрЃћрЃЉрЃљрЃЊ [рЃњрЃљрЃћрЃфрЃљрЃюрЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЦрЃфрЃўрЃљрЃА.](https://www.facebook.com/groups/3969683209738650/permalink/4825174127522883/)

рЃЊрЃдрЃћрЃўрЃЊрЃљрЃю рЃЋрЃћрЃЉ-рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃўрЃА рЃАрЃгрЃљрЃЋрЃџрЃљрЃА рЃЊрЃљрЃЋрЃўрЃгрЃДрЃћрЃЉрЃЌ: рЃерЃћрЃЏрЃЊрЃћрЃњрЃў рЃЎрЃБрЃарЃАрЃў рЃарЃЮрЃЏрЃћрЃџрЃАрЃљрЃф рЃњрЃљрЃўрЃЋрЃџрЃў рЃљрЃарЃўрЃА freeCodeCamp - рЃћрЃА рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃарЃћрЃАрЃБрЃарЃАрЃўрЃљ рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЋрЃћрЃЉ-рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃА рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃљрЃЊ рЃарЃљрЃЊрЃњрЃљрЃю рЃљрЃарЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃЌрЃћрЃЮрЃарЃўрЃБрЃџ рЃЏрЃљрЃАрЃљрЃџрЃљрЃА рЃўрЃЌрЃЋрЃўрЃАрЃћрЃЉ рЃљрЃарЃљрЃЏрЃћрЃЊ рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃАрЃљрЃф рЃљрЃЎрЃћрЃЌрЃћрЃЉ рЃЊрЃљ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџ рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃћрЃЉрЃљрЃАрЃљрЃф рЃўрЃФрЃћрЃю. рЃњрЃљрЃўрЃљрЃарЃћ рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ - [https://www.freecodecamp.org](https://www.freecodecamp.org/) рЃЊрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃў 3-4 рЃЎрЃЋрЃўрЃарЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў рЃБрЃюрЃЊрЃљ рЃњрЃљрЃўрЃљрЃарЃЮ рЃърЃўрЃарЃЋрЃћрЃџрЃў 3 рЃЎрЃБрЃарЃАрЃў.

рЃЎрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃЕрЃљрЃЏрЃЮрЃюрЃљрЃЌрЃЋрЃљрЃџрЃў:

2. JavaScript Algorithms and Data Structures
3. Front End Libraries

рЃћрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌрЃў рЃЎрЃБрЃарЃАрЃћрЃЉрЃў рЃерЃўрЃюрЃљрЃљрЃарЃАрЃЮрЃЉрЃарЃўрЃЋрЃљрЃЊ рЃўрЃњрЃўрЃЋрЃћ рЃЏрЃљрЃАрЃљрЃџрЃљрЃА рЃцрЃљрЃарЃљрЃЋрЃА рЃарЃљрЃф freeCodeCamp - рЃў рЃЌрЃБрЃЏрЃфрЃљ рЃњрЃљрЃфрЃўрЃџрЃћрЃЉрЃўрЃЌ рЃЎрЃљрЃарЃњрЃљрЃЊ рЃ«рЃАрЃюрЃўрЃА рЃљрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃцрЃарЃўрЃА рЃЌрЃћрЃЮрЃарЃўрЃБрЃџ рЃЏрЃ«рЃљрЃарЃћрЃА рЃЊрЃљ рЃЌрЃљрЃю рЃЎрЃљрЃарЃњ рЃърЃарЃЮрЃћрЃЦрЃбрЃќрЃћрЃф рЃњрЃљрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃА рЃарЃўрЃЌрЃљрЃф рЃФрЃљрЃџрЃўрЃљрЃю рЃдрЃўрЃарЃћрЃЉрЃБрЃџ рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃћрЃЉрЃљрЃА рЃЏрЃўрЃўрЃдрЃћрЃЉ.

рЃерЃћрЃЋрЃљрЃ»рЃљрЃЏрЃЮрЃЌ: рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃБрЃЏ 1 рЃЌрЃЋрЃћ рЃњрЃљрЃЦрЃЋрЃА рЃљрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃцрЃарЃўрЃА рЃЊрЃљрЃАрЃљрЃАрЃарЃБрЃџрЃћрЃџрЃЉрЃџрЃљрЃЊ рЃЊрЃљ III рЃћрЃбрЃљрЃърЃќрЃћ рЃњрЃљрЃЊрЃљрЃАрЃљрЃАрЃЋрЃџрЃћрЃџрЃљрЃЊ.

рЃерЃћрЃЏрЃЊрЃћрЃњ рЃћрЃбрЃљрЃърЃќрЃћ рЃЎрЃўрЃЊрЃћрЃЋ рЃБрЃцрЃарЃЮ рЃАрЃљрЃўрЃюрЃбрЃћрЃарЃћрЃАрЃЮ рЃАрЃљрЃ«рЃљрЃџрЃўрЃАрЃЮ рЃЊрЃљ рЃарЃљрЃф рЃЏрЃЌрЃљрЃЋрЃљрЃарЃўрЃљ рЃњрЃљрЃЏрЃЮрЃЊрЃфрЃўрЃџрЃћрЃЉрЃўрЃА рЃЏрЃЮрЃЏрЃфрЃћрЃЏрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃў рЃњрЃћрЃџрЃўрЃА.
